Typography

Helper classes and examples to help with headings, body text, and more.

Fonts

Flexo

The Health Catalyst corporate font. Only to be used for product names displayed in the header

Font Size

The default font of the body is 14px and all other font sizes are based off that using rems or relative ems that are relative the the font size of the html element

Elements

h1. Page Title

 <h1></h1> or <div class="h1"></div>
  • Font-size 2.5rem or ~36px
  • Line Height 2.625em
  • Weight 600
  • Color $text - $offblack

h2. Subtitle

 <h2></h2> or <div class="h2"></div>
  • Font-size 1.5rem or ~22px
  • Line Height 1.5em
  • Weight 400
  • Color $dark-blue

h3. Section Title

 <h3></h3> or <div class="h3"></div>
  • Font-size 1.4rem or ~20px
  • Line Height 1.5em
  • Weight 300
  • Color $gray-500

h4. Mini Header

 <h4></h4> or <div class="h4"></div>
  • Font-size 1.2rem or ~17px
  • Line Height 1.313em
  • Weight 300
  • Color $text - $offblack

p. Paragraph

 <p></p> or <div class="p"></div>
  • Font-size 1.1rem or ~16px
  • Line Height 1.25em
  • Weight 300
  • Color $text - $offblack
a. Anchor <a href="..."></a>
  • Font-size inherit
  • Line Height inherit
  • Weight inherit
  • Color $primary-brand - $blue
code. Code <code></code>
  • Font-size inherit
  • Line Height inherit
  • Weight inherit
  • Color $magneta 
Preformatted + Code
 <pre><code></code></pre>
  • Font-size 1.2rem
  • Line Height 140%
  • Weight inherit
  • Color $text
